# What is this project?

Movies App represent a list of movies, list of categories, search about movie and Save movie in Room Database

1. Kotlin
2. MVVM
3. Jetpack Compose
4. MutableState
5. Use Case
6. Hilt
7. Navigation Components
8. Retrofit2
9. Pagination
10. Room
